# MediSlot

This project uses a Python backend, so it will not open correctly by double-clicking an HTML file.

## Run the website

Option 1:

- Double-click `start_medislot.bat`

Option 2:

1. Open a terminal in this folder
2. Run `python server.py`
3. Open `http://127.0.0.1:8000/`

## Admin login

- URL: `http://127.0.0.1:8000/login.html`
- Default username: `admin`
- Default password: `MediSlot@123`

You can change the admin credentials with environment variables:

- `MEDISLOT_ADMIN_USER`
- `MEDISLOT_ADMIN_PASSWORD`
